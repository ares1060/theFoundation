Examples

Example for using the Blog Service in an PerPedes (html) Template:

  1. {pp:Blog(mode:list, page:2)} <!--Displays page 2 of the blog//-->
  2. {pp:Blog(mode:cat, page:2, cat:test)} <!--Displays just page 2 of category test//-->

Example for using the Blog Service in php:

  1. require_once('_core/theFoundation.php'); //Include PerPedes
  2.  
  3. $sp = new ServiceProvider(); //Init ServiceProvider
  4.  
  5. $sp->view('Blog', array('mode'=>'tag',
  6.    'tag'=>'test',
  7.    'page'=>1));
  8.  
  9. echo $sp->render(); //The Tag Page of the Tag "test" will be rendered at page 1
